# Requisitos do Projeto — Emprega Jacareí

Este documento consolida o levantamento de requisitos do projeto de extensão universitária **Emprega Jacareí**.

---

## Requisitos Funcionais

Os requisitos funcionais descrevem o que o sistema deve permitir que os usuários realizem.

| Código | Requisito Funcional | Descrição | Prioridade |
| :---: | :--- | :--- | :---: |
| **RF01** | Listar vagas disponíveis | O sistema deve exibir uma lista de vagas publicadas pelas empresas. | **Alta** |
| **RF02** | Exibir detalhes da vaga | O usuário deve poder visualizar informações como cargo, empresa, descrição, requisitos e forma de candidatura. | **Alta** |
| **RF03** | Filtrar vagas por área | O sistema deve permitir a filtragem das vagas por área de atuação. | **Alta** |
| **RF04** | Ordenar vagas cronologicamente | As vagas devem ser exibidas em ordem cronológica, priorizando as mais recentes. | **Média** |
| **RF05** | Realizar candidatura rápida | O candidato deve poder se candidatar por meio de um formulário simples, sem criar conta. | **Alta** |
| **RF06** | Anexar currículo | O formulário de candidatura deve permitir o envio de currículo em PDF, DOC ou DOCX. | **Alta** |
| **RF07** | Enviar candidatura à empresa | Após o envio do formulário, a empresa deve receber os dados do candidato e o currículo. | **Alta** |
| **RF08** | Cadastrar vaga | Empresas ou administradores devem conseguir cadastrar novas vagas. | **Alta** |
| **RF09** | Editar vaga | Empresas ou administradores devem poder alterar informações de uma vaga já cadastrada. | **Média** |
| **RF10** | Remover ou desativar vaga | O sistema deve permitir remover ou desativar vagas expiradas ou preenchidas. | **Média** |
| **RF11** | Validar campos obrigatórios | O sistema deve impedir o envio de formulários incompletos. | **Alta** |
| **RF12** | Exibir mensagem de confirmação | Após a candidatura, o sistema deve informar que o envio foi realizado com sucesso. | **Média** |
| **RF13** | Gerenciar categorias de vagas | O sistema deve permitir organizar vagas por áreas ou categorias. | **Baixa** |
| **RF14** | Painel administrativo | O sistema deve possuir uma área administrativa para gerenciamento de vagas e candidaturas. | **Média** |
